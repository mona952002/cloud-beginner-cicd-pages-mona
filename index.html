What is CI/CD?

CI/CD refers to Continuous Integration and Continuous Deployment, two practices used to improve software delivery. Developers can automatically test and deploy their applications whenever new changes are pushed. This approach increases efficiency and helps teams release reliable software faster.

My Pipeline

When new code is pushed to the repository, a GitHub Actions workflow runs automatically. It verifies the project files and deploys the website to GitHub Pages. As a result, any update made to the repository appears on the live website after the workflow finishes.
